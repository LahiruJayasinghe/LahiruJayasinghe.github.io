Project Overview

Industrial machinery naturally deteriorates over time, and repairing a faulty machine can be extremely expensive. To prevent such failures in advance, a monitoring system is essential. This system tracks the machinery's decay cycle and signals when maintenance is needed.

Solution

Implement an Remaining Useful Life (RUL) estimation model that continuously predicts the machine's RUL at each time step throughout its operational life. This solution has been successfully implemented and deployed for Keysight Technologies Singapore.

Work summary

  • Participated in customer meetings about the features and requirement of the project.
  • Data cleaning, normalizing, and visualizing on PCA/TSNE space using pandas and scikit-learn to identify inherent clusters and behaviors.
  • Implement anomaly detection pipeline using dimensional reduction and clustering (dbscan and kmeans).
  • Proof of concept (POC) model implementation using TensorFlow on similar online datasets by combining CNN and LSTM with data augmentations to predict RUL.
  • Train and evaluate the model on customer data while fine-tuning the hyperparameters.
  • Implement the deployment for the final solution (combining anomaly detection and fine-tuned model) as a Docker image for the project delivery.
  • Publish a research paper with Keysight Technologies about the new findings of the project in a flagship IEEE conference which can be found online here.
