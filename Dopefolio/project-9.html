Project Overview

Enhance security by developing a system capable of detecting the presence and approach direction of drones using an antenna array.

Solution

Utilize micro-Doppler signatures collected from the antenna array as data. Train a denoising autoencoder, and employ the encoder with a classifier to accurately identify the direction from which unmanned aerial vehicles (UAVs) are approaching.

Work summary

  • Data collection (micro-Doppler signatures) using special antennas.
  • Formulated the problem as a classification task and designed evaluation methods to assess robustness.
  • Robustness was achieved by denoising autoencoders while introducing sparsity constraints to help the neural network extract meaningful features for use in the classification stage.
  • Implementations were done using TensorFlow, Python, Pandas, GNU-radio, and the solution was deployed on an antenna platform.
  • Published a research paper about the new findings of the project in an IEEE conference (link).
