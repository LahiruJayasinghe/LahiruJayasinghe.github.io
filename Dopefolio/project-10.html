Project Overview

The goal is to devise a system capable of real-time detection and counting of individuals entering and exiting a specific location.

Solution

Utilize Edge-AI concepts, focusing on object detection, and model pruning to make it suitable for edge devices.

Work summary

  • Data collection (video feed) and processing them as images using OpenCV.
  • Implemented the object detection model using MobileNet-SSD in TensorFlow to detect people from the bird's-eye view of the camera.
  • Worked with another engineer to convert the model to Caffe and then deployed it on the Intel Movidius neural compute stick.
  • Performed model pruning while maintaining good enough accuracy to reduce the size of the model, decrease the inference time, and make it fit into an edge device.
  • Published an IEEE journal paper about the new findings of a pruning approach (link).
