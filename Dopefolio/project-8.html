Project Overview

Objective is to develop a system to immediately respond to cleaning requirements which will help in improving the Quality of Service (QoS) while reducing costs incurred due to periodic cleaning.

Solution

Implement a image classifier to identify different clenliness levels on washrooms and notify if the clenliness level is low consecativly.

Clientele

Keysight Technologies.

Work summary

  • Participated in customer project meetings to discuss project deliverables.
  • Implemented PCA-based data augmentation technique (similar to data augmentation used in the AlexNet paper) using OpenCV and scikit-learn to enhance dirtiness in images.
  • Implemented feature visualization to select the backbone network that best explains the customer data for transfer learning.
  • Model implementation using TensorFlow on customer data to detect dirtiness by classification.
  • Publish an IEEE journal paper about the approach of the project here.
