Project Overview

Detect twitter bots and classify them into spam/traditional/social bots by implementing a model using labeled dataset and use it to analyze unlabeled dataset

Data

The labeled dataset was an aggregated dataset having ~20 million tweets from different data sources. The unlabeled dataset is collected from 2020 twitter competition having ~160 million tweets and ~50GB large

Work summary

  • Conduct project planning and literature reading to understand SOTA approaches for Twitter bot detection.
  • Implement data aggregation, data cleaning, data structure conversions (associative arrays), data storing using Scala/Spark and Json file stores.
  • Implement tweet similarity features using MinHash with LSH and many more tweet and user profile-related features in Scala.
  • Implement SVM, XGBoost, Random Forest, Decision trees, Logistic Regression, and MLP models using PySpark ML library on Databrick clusters on Azure.
  • Perform hyperparameter tuning with Cross-validation to select the best model to perform data analysis on unlabeled data.
  • Perform data and analysis using statistical information, PCA, kmeans, and network (graph) visualization on unlabeled data.
