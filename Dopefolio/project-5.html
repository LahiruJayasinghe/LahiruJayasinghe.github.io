Project Overview

Airlines often carry surplus inflight meals that go to waste after each trip. Our solution identifies and provides statistics on this food waste, enabling informed decisions to reduce waste and enhance the profitability of inflight catering services.

Solution

Given an image, detect and count different type of inflight meals using object detection models and provide statistics based on these model results.

Clientele

SIA, Ethihad, etc.

Work summary

  • Implement train/test data pipelines using tf.data and TFRecord.
  • Implement Haze removal technique to preprocess images.
  • Evaluation of STOA feature extractors with respect to speed and accuracy.
  • Implement the object detection solution in TensorFlow and integrated rule-based post processing to provide metadata for analytics.
  • Work with the customer success team to verify the model accuracies for each class.
  • Implement the inference pipeline which downloads the data from GCP by referring to the respective database tables, then perform the inference and update the results back to the database.
  • Deploy the system on GCP with the help of Docker.
