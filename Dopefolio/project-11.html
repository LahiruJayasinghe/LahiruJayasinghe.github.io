Project Overview

Detect malware based on features extracted from the API calls

Solution

Utilize Edge-AI concepts, focusing on object detection, and model pruning to make it suitable for edge devices.

Data

The shape of each data sample is (LENGTH, FEATURES). It has 102 FEATURES, but the LENGTH is at most 1000 and it is not fixed. All the data is labeled as a binary classification

Work summary

  • Perform histogram analysis for the data sample LENGTH that helps to make it a fix length instead of padding them.
  • Perform data normalization and PCA using Scikit-Learn.
  • Implement a random sampling batch generator to balance out the classes in a batch to tackle class imbalance in data.
  • Implement a model with multi-layered GRU cells with dropout regularization and sigmoid function with binary cross-entropy as the loss function.
  • Model achieved ~98% accuracy in the public leaderboard and ~97% in the private leaderboard in Kaggle platform.
